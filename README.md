# Simulacre: Performance test 3rd module
## 

## Dependencies
- Node.js
- JSON Server
## Instalation
- clone the repository
```bash
    git clone https://github.com/SrLampi1001/simulacro_prueba_desempeno_m3.git
```
- Install json server
```bash
    npm install json-server
```
- Initialize db.json in localhost:3000
```bash
    npx json-server database/db.json
```
